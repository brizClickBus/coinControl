Termos de uso e política de compartilhamento de dados

Coin Control
O CoinControl respeita a privacidade dos seus usuários e se esforça para proteger seus dados pessoais.
Esta política de compartilhamento de dados descreve como e quando compartilhamos informações com terceiros.

Coleta e Uso de Dados
Ao criar uma conta no CoinControl, coletamos informações pessoais básicas, como nome, sobrenome, endereço de e-mail e senha.
Essas informações são usadas exclusivamente para fornecer o serviço do CoinControl aos nossos usuários e nunca são compartilhadas com terceiros
a menos que seja necessário para cumprir uma solicitação do usuário ou exigência legal.

Compartilhamento de Dados
O CoinControl não compartilha informações pessoais dos usuários com terceiros, a menos que sejam necessárias para cumprir uma solicitação do usuário ou exigência legal.
Podemos compartilhar informações não pessoais, como estatísticas de uso agregadas, com terceiros para fins de análise e melhorias do serviço.

Segurança de Dados
O CoinControl toma medidas razoáveis para proteger os dados pessoais dos usuários contra acesso não autorizado, uso indevido ou divulgação.
Usamos tecnologias padrão da indústria, como criptografia e autenticação de dois fatores, para proteger as informações pessoais dos usuários.

Alterações na Política de Compartilhamento de Dados
O CoinControl pode atualizar esta política de compartilhamento de dados periodicamente para refletir mudanças na forma como coletamos, usamos ou compartilhamos informações pessoais.
Ao continuar a usar o serviço após a publicação de qualquer atualização, o usuário concorda com os termos revisados. br
